# Zukunftsmusik

This is the survey data for the Zukunftsmusik Concert (https://kreativ.institute/de/events-und-seminare/zukunftsmusik)


Questionnaire.pdf is a printout of the survey

limesurvey_survey_357913-2.lss is the LimeSurvey backup of the actual survey that can be imported and setup for a new survey

SurveyResults.xlsx contains the anonymized data
